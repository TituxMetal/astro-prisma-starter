# Astro Course Files

This is the starter project for the Astro course from
[Chris Pennington (Coding in Public)](https://www.youtube.com/@CodinginPublic/videos)

It uses the following technologies:

- Astro
- React
- Tailwind CSS
- TypeScript
- ESLint
- Prettier

## Development

```bash
yarn dev
```

## Production

```bash
yarn build
yarn preview
```
